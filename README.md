## Installation

```bash
$ git clone 
#
$ yarn 
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn  start:prod
```

## Test

```bash
# unit tests
$ yarn  test

# e2e tests
$ npm run test:e2e


```
